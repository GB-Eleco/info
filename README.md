# 学校信息整合

这是一个可以直接部署到 GitHub Pages 的静态站点，用于展示高校学院报名信息。

## 功能

- 左侧按高校档次和省份筛选
- 右侧展示学院条目
- 报名未开始显示灰色
- 报名已结束显示划线
- 每个条目都是可点击链接

## 数据结构

当前数据放在 `data.js` 里，作为简单易用的前端“数据库”。

字段包括：

- 学校名称
- 学院名称
- 所在省份
- 报名开始时间
- 报名截止时间
- 学校档次
- 强/弱 com
- 是否机试
- 学院链接

## 部署到 GitHub Pages

1. 把这些文件推送到仓库根目录。
2. 在 GitHub 仓库的 Settings 里打开 Pages。
3. Source 选 `Deploy from a branch`。
4. Branch 选 `main`，目录选 `/root`。
5. 保存后等待几分钟即可访问。

## 后续扩展

如果你后面想把数据真正接到数据库后端，可以把 `data.js` 替换成 API 请求，再接入 Supabase、Firebase、Vercel 或自建后端。
